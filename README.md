## What Does Taggbox Actually Do?

At its core, Taggbox is a social media aggregator — it pulls in content from wherever your customers are posting and turns it into something you can use across your marketing.

A few things it does particularly well:

**Social Media Widgets**: Embed feeds from Instagram, TikTok, Twitter/X, YouTube, LinkedIn, Pinterest, RSS, and more on any website or CMS (WordPress, Shopify, Wix, Squarespace, Webflow, etc.).

**Review Widgets**: Pull in Google Reviews, Facebook Reviews, Yelp, Airbnb reviews and display them beautifully — which is basically a trust machine for any product page or service landing page.

**Shoppable UGC**: Tag products directly in Instagram and TikTok posts so visitors can go from "ooh I want that" to checkout without leaving your site. **Analytics**: Track widget views, post clicks, active users, and sentiment analysis — all in a dashboard you can actually read.
